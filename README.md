# FUTURE_DS_01 – Sales Data Analysis

## Project Overview

This project analyzes retail sales data to uncover business insights, sales trends, and customer behavior.

The goal of this analysis is to help businesses understand:

• Which product categories generate the most revenue  
• Which regions perform best in sales  
• Monthly sales trends over time  
• Who the top customers are  

This type of analysis is commonly performed by data analysts to support business decision-making.

---

## Dataset

Dataset used: Superstore Sales Dataset

The dataset contains information about:

- Order Date
- Customer Name
- Product Category
- Region
- Sales
- Other transactional details

---

## Tools Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Data Visualization

---

## Steps Performed

### 1. Data Loading
The dataset was loaded into Python using Pandas.

### 2. Data Cleaning
- Checked dataset structure
- Handled missing values
- Converted date columns to proper datetime format
- Removed duplicate entries

### 3. Exploratory Data Analysis

The following analyses were performed:

#### Sales by Category
Analyzed which product categories generated the highest sales.

#### Sales by Region
Compared performance across different regions.

#### Monthly Sales Trend
Analyzed how sales changed over time using monthly aggregation.

#### Top Customers
Identified the customers who generated the highest revenue.

---

## Key Insights

• Technology category generated the highest sales.

• West region recorded the highest total sales.

• Sales tend to spike during the end of the year (November–December).

• A small group of customers contribute significantly to total revenue.

---

Charts created during analysis include:

---

## Business Recommendations

Based on the analysis:

• Focus marketing on high-performing categories such as Technology.

• Strengthen sales strategies in lower-performing regions.

• Prepare inventory and marketing campaigns before peak months.

• Build loyalty programs targeting high-value customers.
